# Flight Simulator 2020 Web Controller

Using an webapp to control you Flight simulation

# How to run

## 1. Run server

from `/server` folder

- [ ] Requirements file

```bash
python -m flask --app main run
```

## 2. Run webapp

Install Quasar CLI

```bash
npm i -g @quasar/cli
``` 

Run Quasar dev server from `/client` folder

```bash
npx quasar dev
```
